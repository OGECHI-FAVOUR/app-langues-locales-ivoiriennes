# Application d'apprentissage des langues locales ivoiriennes

## Description
Application interactive permettant l'apprentissage des langues locales 
ivoiriennes, intégrant des fonctionnalités innovantes pour faciliter 
l'accès à la culture et aux langues nationales.

## Fonctionnalités
- Apprentissage interactif des langues locales
- Interface simple et accessible
- Contenu culturel intégré

## Technologies utilisées
- HTML / CSS / JavaScript
